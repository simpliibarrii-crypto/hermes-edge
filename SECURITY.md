# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please email **bclerjuste@gmail.com** with details of the issue.

Do **not** open public GitHub issues for security vulnerabilities.

You should receive a response within 48 hours. If the issue is confirmed, a patch will be released as soon as possible depending on complexity.
