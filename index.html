Edge intelligence, measured.

Hermes Edge is a research runtime for local AI agents across phones, laptops, and edge boxes. It routes deterministic tools first, keeps model execution local when supported, and ties public claims to reproducible benchmark evidence.

Designed to spend compute carefully.

Hermes treats larger model calls as a later route, not the default reflex.

01 · Tool first

Deterministic work stays deterministic

Arithmetic, formatting, retrieval, and local utilities can resolve before an expensive model is loaded.

02 · Local policy

Route by device evidence

Profiles describe supported runtime, memory, backend, and model paths without claiming hardware performance that has not been measured.

03 · Auditable output

Judge terminal artifacts

Workflow benchmarks compare canonical output contents, not cosmetic directory layouts or task names.

Benchmark contract

Evidence before acceleration claims.

A design can be promising. A performance statement needs a device, backend, workload, and reproducible result.

This hosted page documents the project. It does not execute the local model and is not a clinical or device-performance validation.
